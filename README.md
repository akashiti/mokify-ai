# mockify-ai
Mockify is an AI-driven interview practice website designed to help users improve their resume quality, communication skills, and interview performance.
It provides real-time scoring, detailed feedback, and personalized suggestions so users can confidently prepare for real job interviews.

-- Key Features

->AI Mock Interviews with instant performance analysis

->Resume Score & Interview Score highlighting strengths and weaknesses

->Honest, actionable feedback for improvement

->Clean, user-friendly dashboard for tracking progress

->Mockify makes interview preparation smarter, faster, and more effective
